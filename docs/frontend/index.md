## 学习路线

```
HTML/CSS 基础 → JavaScript 核心 → JavaScript 进阶 → TypeScript → 框架(Vue/React) → SSR 框架 → 跨端开发
```

::: tip 建议
面试重点关注：
- **JavaScript 核心**：闭包、原型链、Event Loop、异步编程
- **JavaScript 进阶**：Proxy/Reflect、Generator、函数式编程、设计模式
- **框架原理**：响应式系统、虚拟 DOM、组件通信
- **SSR 框架**：Next.js、Nuxt.js 的渲染模式、数据获取、SEO 优化
:::
