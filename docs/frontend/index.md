## 学习路线

```
HTML/CSS 基础 → JavaScript 核心 → TypeScript → 框架(Vue/React) → 跨端开发
```

::: tip 建议
面试重点关注 JavaScript 核心（闭包、原型链、Event Loop）和框架原理（响应式、虚拟 DOM）
:::
