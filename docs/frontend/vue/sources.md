## 结果摘要
- Exa 抽检结果以官方文档与社区最佳实践为主，已用于补齐高频题与最佳实践要点。

## 高频题清单（摘要）
- Vue 2 与 Vue 3 响应式系统的差异与实现原理。
- Composition API 与 Options API 的取舍与适用场景。
- 组件通信方式与跨层级依赖注入（provide/inject）。
- `v-if` 与 `v-show` 的性能权衡。

## 最佳实践要点（摘要）
- 优先使用 Composition API 组织复杂逻辑，避免过度复用 mixin。
- 组件拆分与按路由/功能分层，保持单一职责。
- `v-for` 必须设置稳定 key，避免状态错乱。
- 合理使用 Pinia/Vuex，避免全局状态膨胀。
- 路由与组件按需加载，优化首屏性能。
