## 学习路线

```
Git 基础 → Linux 命令 → Docker 容器 → Kubernetes → CI/CD
```

::: info 面试重点
- Git: 分支策略、合并冲突、回滚操作
- Docker: 镜像构建、容器编排、网络模式
- HTTP: 状态码、缓存机制、HTTPS 原理
- TCP: 三次握手、四次挥手、拥塞控制
:::
