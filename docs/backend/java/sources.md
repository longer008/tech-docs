## 结果摘要
- Exa 抽检结果以官方文档与社区最佳实践为主，已用于补齐高频题与最佳实践要点。

## 高频题清单（摘要）
- JDK/JRE/JVM 区别与运行机制。
- 线程池参数与拒绝策略。
- JMM 内存模型与可见性/有序性。
- `HashMap`/`ConcurrentHashMap` 差异。
- GC 基础与分代回收。

## 最佳实践要点（摘要）
- 合理配置线程池与队列。
- 避免不必要的同步与锁竞争。
- 使用 `try-with-resources` 管理资源。
- 关注 LTS 版本差异与升级风险。
