# 后端开发

后端开发技术栈涵盖 Java、Node.js、Python 三大生态。

## 模块概览

| 分类 | 技术 | 链接 |
|------|------|------|
| Java 生态 | Java SE/JUC, Spring Boot, Spring Cloud, MyBatis | [查看](/backend/java/java-core-interview) |
| Node.js 生态 | Runtime, Express/Koa/NestJS | [查看](/backend/nodejs/nodejs-runtime-interview) |
| Python 生态 | Python 基础, Django, FastAPI | [查看](/backend/python/python-core-interview) |

## 学习路线

### Java 路线
```
Java 基础 → JUC 并发 → Spring Boot → Spring Cloud → MyBatis
```

### Node.js 路线
```
Node.js Runtime → Express/Koa → NestJS → 微服务
```

### Python 路线
```
Python 基础 → Django/FastAPI → 异步编程 → 微服务
```

::: tip 建议
面试重点关注并发编程、框架原理、微服务架构
:::
