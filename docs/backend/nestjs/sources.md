## MCP 查询记录

### 查询信息

- **查询时间**：2025-02-04
- **Library ID**：`/websites/nestjs`
- **代码示例数量**：2103 个

### 关键发现

1. **核心概念**：
   - 模块化架构（Module、Controller、Provider）
   - 依赖注入（IoC 容器）
   - 装饰器（Decorator）

2. **请求处理流程**：
   - 中间件 → 守卫 → 拦截器（前） → 管道 → 方法 → 拦截器（后） → 异常过滤器

3. **最佳实践**：
   - 使用 Fastify 提升性能
   - 启用压缩和缓存
   - 使用连接池优化数据库
   - 避免阻塞事件循环

4. **微服务支持**：
   - 支持 TCP、Redis、NATS、MQTT、Kafka 等传输层
   - 提供消息模式和事件模式
   - 支持混合应用（HTTP + 微服务）

5. **测试策略**：
   - 单元测试（Jest + DI mocking）
   - E2E 测试（Supertest）
   - 使用 TestingModule 配置依赖

### 学习路线

**初级（1-2 周）**：
- 掌握基础概念（Module、Controller、Provider）
- 学习依赖注入和装饰器
- 实现简单的 CRUD 应用

**中级（2-4 周）**：
- 掌握管道、守卫、拦截器
- 学习数据库集成（TypeORM/Mongoose）
- 实现认证授权（JWT）

**高级（1-2 个月）**：
- 掌握微服务架构
- 学习性能优化
- 实现复杂的企业级应用
